# iod-blog-team

Example Express Api using Sequelize and MySQL.

Some example views with Handlebars.

Topics:
- Validation with express-validator
- Logging with Winston and Morgan
- Error handling
- Swagger with JsDoc
- Handlebars with Bootstrap
- Docker and Docker Compose
- Github Actions
